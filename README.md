# Site Personal - Airimitoaie Fabian-Silviu

Site personal creat cu React, Vite și Tailwind CSS.

## Tehnologii

- React 18
- Vite
- Tailwind CSS
- React Router
- jQuery (pentru galerie 3D)
- Lucide React (iconuri)

## Instalare

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Vezi `DEPLOYMENT.md` pentru instrucțiuni detaliate de deployment.

## Caracteristici

- Design elegant cu paletă de culori premium
- Responsive design
- Galerie cu carusel 3D circular
- Suport multi-limbă (Română/Engleză)
- Navigare fluida între pagini
